Description des unités d’enseignement

    L'option Programmation Créative permet de vous professionnaliser et d’acquérir les bases pour une production autonome de créations numériques. Cette option vous offre une initiation à la programmation informatique à travers des langages simples permettant la création de visuels et de sites web. L'accent sera mis sur la réalisation de projets personnels orientés vers les humanités numériques et destinés à développer votre autonomie et votre esprit critique vis à vis du monde numérique contemporain.

  • SEMESTRE 3:

    • Représentation numérique de l’information : 3 ECTS, 24h, vous verrez comment sont représentés les caractères, les textes, les images et les videos pour pouvoir les manipuler.

    Programmation créative 1 : 3 ECTS, 24h, ce cours constitue une initiation à l'environnement de programmation créative Processing. Les bases du langage de programmation seront abordés.

  • SEMESTRE 4:

    • Langages du Web : 3 ECTS, 24h, vous souhaitez publier des pages Web ? vous apprendrez à comprendre, connaître et manipuler les langages pour publier sur le Web, en particulier les langages HTML 5 et CSS.

    Programmation créative 2 : 3 ECTS, 24h, dans ce cours vous apprendrez à construire des réalisations numériques animées avec l'environnement Processing.

  • SEMESTRE 5:

    • Introduction aux bases de données : 3 ECTS, 24h, vous souhaitez créer un site Web ? Il faut organiser les données de votre site dans une base de données relationnelle. Vous étudierez les fondements des bases de données et le langage d’interrogation SQL.

    Programmation créative 3 : 3 ECTS, 24h, dans ce cours vous apprendrez à construire des représentations de données dédiées à un propos spécifique (visualisation des données).

  • SEMESTRE 6:

    • Gestion de contenu et Web : 3 ECTS, 24h, vous devez créer un site complet ? vous aurez ici une présentation complète des outils de gestion de contenus pour le Web pour les comprendre, les manipuler et savoir choisir le(s) meilleur(s) outil(s) selon vos besoins.

    Programmation créative 4 : 3 ECTS, 24h, dans ce cours vous apprendrez à construire des réalisations numériques interactives , qui répondent aux usages d'un utilisateur.
